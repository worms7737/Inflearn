## 강의에서 다루는 전체 환경

**어떤 환경으로 설치가 되고 진행될까?(흐름)**

### 개발환경

*내 PC가 개발 툴이 된다.* <br>
-> 개발 툴/도구 -> IntelliJ <br>
-> 개발 프레임워크 -> Spring boot <br>
-> 개발 언어/패키지 -> OpenJDK <br>

### CI/CD 환경

Guest OS -> Rocky Linux <br>
컨테이너 빌드 -> Docker <br>
빌드/배포 도구 -> Jenkins <br>

스크립트 문으로 해당 설치 내용 수행

### 인프라 환경

컨테이너 오케스트레이션 -> Kubernetes <br>
컨테이너 런타임 -> ContainerD
